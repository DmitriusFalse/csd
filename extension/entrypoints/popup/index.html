❓ Помощь

Расширение для скачивания моделей с Civitai через локальный сервер.

Как использовать:

  1. Запусти csd-bridge.exe (сервер)
  2. Нажми на зелёную точку 🟢 в попапе — проверить подключение
  3. Зайди на civitai.com или civitai.red
  4. Нажми фиолетовую кнопку "Download via CSD" на странице модели

Настройки ⚙:

  • Порт сервера — должен совпадать с портом в config.yaml
  • API-ключ — опционально, можно указать в config.yaml
  • Путь из LM — автоматически определяет корень для каждого типа модели через Lora Manager

Lora Manager:

  • Укажи URL (по умолч. http://127.0.0.1:8188)
  • Галка «Путь из LM» — путь для LoRA/LyCORIS/Checkpoints берётся из LM
  • Если LM не отвечает или тип не поддерживается — используется путь по умолчанию

Где искать:

  • Логи сервера: кнопка 📄 в настройках
  • Config: открой config.yaml рядом с csd-bridge.exe
